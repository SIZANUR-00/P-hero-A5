> In JavaScript we use `var`, `let`, and `const` to declare variables. `var` is the older way and it is function scoped. It can be redeclared and its value can also be changed. `let` was introduced later and it is block scoped, so it only works inside the block where it is declared. We can change its value but we cannot redeclare it in the same scope. `const` is also block scoped but its value cannot be changed after it is declared, so it is mainly used when we donвЂ™t want the value to change.

> The spread operator `...` is used to expand the elements of an array or object. It is mainly used when we want to copy an array, combine multiple arrays, or copy values from one object to another. It makes the code shorter and easier to understand.

3пёЏвѓЈ **What is the difference between map(), filter(), and forEach()?**

> `map()`, `filter()`, and `forEach()` are array methods in JavaScript. `map()` is used when we want to create a new array by modifying each element of the original array. `filter()` is used when we want to get elements that match a certain condition. `forEach()` is used to run a function for every element in the array, but it does not return a new array.
